#Adventure Pack

Adventure Pack is a DM's Toolkit built on web technologies.

##Installation

``` bash
# install deps
npm install

# build dist files
npm run build

# or watch files to be built
npm run watch

# serve locally via your own source or...
npm install http-server
http-server
```
